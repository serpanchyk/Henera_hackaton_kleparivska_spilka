## Місія лідера: прямий політ

Проста автономна місія дрона-лідера: зліт на 1 м над платформою, політ
прямо вперед на 12 м, утримання позиції, посадка. Курс і висота захоплюються
динамічно після виходу в стабільний hover (без жорсткої прив'язки до напрямку).

Швидкість лідера обмежена `LEADER_CRUISE_SPEED_M_S` (3 м/с), щоб не випереджати
фолловерів. Параметри редагуються константами на початку `mission_launch.py`.

### Запуск з нової консолі:
```sh
cd resources/scripts/missions/straight_line/
python3 mission_launch.py
```

Ця місія також запускається автоматично через `solution_launch.py`
(t=20s після старту Gazebo).
