## **Увага! Файли репозиторію можуть змінюватися, робота над ними ще ведеться**
---- 
### ---*Доробляється маршрут дрона-лідера*---
----

#### Містить Файли проекту в теці "resources"

#### Файли автоматично копіюються під час запуску скрипта project_setup.sh

#### Увага! скрипт має запускатися тільки після розгортання PX4, тому що містить шляхи до його тек. 

#### Інструкція з встановлення міститься в *README_setup.md*

В цьому файлі розписано розгортання середовища.

#### Запуск скрипта:
```sh
./project_setup.sh
```

#### Перед запуском проекту:
```sh
cd ~/PX4-Autopilot/
source ~/falcon_gaze/resources/scripts/px4_gz_setup.sh
```

#### Проект запускається за допомогою python-3 скрипта
*"swarn_launch.py"* через **ROS2**. Перед запуском необхідно прописати команду:
```sh
source /opt/ros/humble/setup.sh
```
*Увага! Якщо у Вас інша версія ROS2 то необхідно замінити "humble" на Вашу версію, наприклад "jazzy", що виглядатиме як > source /opt/ros/jazzy/setup.sh
*

#### Запуск скрипта:
```sh
cd ~/PX4-Autopilot/
ros2 launch ~/falcon_gaze/resources/scripts/swarn_launch.py
```
Завантажиться **PX4** та підвантажить світ **Gazebo** в якому будуть чотири дрони. Дрон-лідер та три дрони-послідовники. У всіх дронів є імітація led маячків представлена у вигляді яскраво-зелених сфер. Також у кожного дрона є камера.

Для того, щоб зупинити симуляцію та закрити вікно:
У консолі в якій запущено *PX4*
**Ctrl+c**
або в іншій 
```sh
pkill -9 px4
```


#### У кожен новий термінал в якому буде запускатися ROS2 необхідно ввести
```sh
source /opt/ros/humble/setup.sh
```
#### У іншій консолі прописуємо:
```
cd ~/falcon_gaze/resources/scripts/
```
#### та запускаємо тренувальний скрипт
```sh
python3 mission_launch.py
```
